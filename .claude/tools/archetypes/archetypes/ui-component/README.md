# UI Component Archetype

Create accessible, themeable UI components with modern frontend frameworks.

## Features

- React, Vue, Svelte, or Web Components support
- Multiple styling options (CSS Modules, Styled Components, Tailwind, Vanilla CSS)
- TypeScript support
- Accessibility (ARIA, keyboard navigation)
- Dark mode and theming
- Storybook integration
- Comprehensive testing setup

## Usage

```bash
archetype create ui-component -o my-ui-library
```

## Example Output

A production-ready component library with proper TypeScript types, accessibility features, and comprehensive documentation.
