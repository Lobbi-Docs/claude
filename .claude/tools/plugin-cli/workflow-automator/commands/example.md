# /example

Example command for workflow-automator

## Usage

```
/example [options]
```

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--verbose` | Show detailed output | false |
| `--dry-run` | Preview without executing | false |

## Workflow

1. **Initialization**: Prepare environment
2. **Execution**: Run main workflow
3. **Validation**: Verify results
4. **Cleanup**: Finalize and report

## Examples

```bash
# Basic usage
/example

# With options
/example --verbose
```
